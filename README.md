Weather App 🌤️
Ek simple aur interactive Weather App, jise HTML, CSS, aur JavaScript ka use karke banaya gaya hai. Yeh app user ko kisi bhi city ka current weather update dikhati hai.

🚀 Features
Real-time Updates: City ka naam search karte hi temperature aur weather conditions dikhata hai.

Modern UI: Clean aur user-friendly interface.

Responsive Design: Mobile aur Desktop dono par sahi se work karta hai.

🛠️ Technologies Used
HTML5: Structure ke liye.

CSS3: Styling aur animations ke liye.

JavaScript: API call aur functionality ke liye.

OpenWeatherMap API: Weather data fetch karne ke liye.

💻 How to use
Is repository ko clone karein:

Bash
git clone https://github.com/kumarmanjeet7479-jpg/WeatherApp.git
index.html file ko apne browser mein open karein.

City ka naam type karein aur search button par click karein.
